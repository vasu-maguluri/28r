HTML Concepts Introduction

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables. Like any language, HTML has syntax. This means that, in order for your code to be considered “well formed” (the HTML equivalent of grammatically correct), you must follow certain foundational rules. Failing to follow these rules means that a computer will not understand your HTML code.

Element

Element is combination of tag,Attribute and coontent given inside the tag.

Types Of elements

1)Block Level Elements

The Elements which occupies full width and creates a new line is known as block level element

2)Inline-block Elements

The Elements which occupies only required width and doesnot create a new line and accepts width, height, margin and padding properties of CSS Ex: img iframe tag button tag

3)Inline Elements

The Elements which occupies only required width and does not create a new line and does not accepts width, height, margin-top and margin- bottom, padding-top and padding-bottom properties of CSS is known as inline elements Ex: a, span, formatting tags
